常见问题 (FAQ)

烦村黄昏整合包需要什么客户端环境？

答：烦村黄昏整合包需要的客户端环境：JDK（Java） 17+、Minecraft 1.19.2、Forge 43.4.1+。

下载单独的整合包、模组后怎样安装？

答：
1、整合包安装：PCL/PCL-CE启动器：直接把下载好的整合包文件拖进启动器窗口即可。HMCL/FCL启动器：下载 -> 整合包 -> 安装整合包 -> 导入本地整合包 -> 选下载好的整合包文件。
2、模组安装：PCL/PCL-CE启动器：直接把下载好的模组文件拖进启动器窗口即可。HMCL/FCL启动器：游戏 -> 游戏管理 -> 模组管理 -> 模组文件夹 -> 将模组文件放入打开的文件夹中。

烦村黄昏整合包怎么玩？

答：在Minecraft游戏中，加载烦村黄昏整合包后，核心玩法基本还跟原生模式一样，不同之处在于：烦村黄昏有原创的装备、物品、技能、生物、NPC。玩家可以学习各种技能与NPC生物激烈战斗、与其它玩家热血PK。

整合包常用操作按键有哪些？

答：进入游戏后，默认游戏操作按键如下（都可在设置里的按键绑定中修改）：
R：开启、关闭战斗模式或查找物品配方；
F：切换主手副手武器；
E：物品栏；
ALT：战斗翻滚；
鼠标侧键1：翻滚；
鼠标侧键2：技能；
鼠标右键：防御；
鼠标左键：攻击；
Z：投掷末影珍珠。

怎样进入烦村黄昏"竞技场"模式？

答："竞技场"模式只有战斗，别无其它。玩家可以单独或组队向竞技场中的NPC生物发起挑战。进入竞技场后，地图中会自动刷新出敌对NPC生物，当被击杀后，会再次出现更为强大的生物，以此类推，直到击杀数十种生物，完成通关。
注1：NPC生物除了会使用技能攻击，还会使用末影珍珠、吃金苹果等，很强的。
注2：每次进入竞技场时，玩家会获得不同的装备、武器，可拥有不同强度的技能哦。
注3：如果独自打不过NPC生物，可以开启服务器并邀请朋友组队，以多打少，一起在竞技场挑战NPC。

游戏中，整合包的NPC生物太强了、打不过怎么办？

答：可以开启作弊模式。在游戏中，输入：/gamerule domob...此时会有提示。
例如：domobusefireball为是否启用火焰弹、domobuseenderperal为是否启用末影珍珠，等等，以增强自己。
注："生存模式"、"竞技场模式"，都可使用此方法。

怎样调整烦村黄昏NPC生物的格挡率？

答：输入/gamerule clashpercent+数字。
数字为1时，格挡概率为10%、数字为5时，格挡概率为50%，以此类推，数字为0时，格挡概率为0%、数字为10时，挡概率为100%。
注："生存模式"、"竞技场模式"，都可使用此方法。

整合包中有哪些二创武器？

答：烦村黄昏二创武器丰富多样，以下是部分武器介绍：
战神之刃类：有TNT战神之刃、雷电战神之刃、冰火战神之刃、战神之刃、双大剑战神之刃、火焰战神之刃、绿宝石战神之刃等，部分武器蹲下右键可释放技能。
蛇刃类：包括未解放蛇刃、解放蛇刃、战蛇之刃、超级解放蛇刃。
影锤：Shift+右键可让玩家原地飞起来。
影镰：能召唤末影龙。
黑曜石长枪：动作和奇迹武器里的痛苦之枪相似。
暗影之镰：属于玩家投稿的概念性DIY武器，只能在创造模式下获取。
某些武器还有格外功能，如：黑盾有格挡技能；红宝剑有星火燎原技能；盔影胸甲有突刺技能。
